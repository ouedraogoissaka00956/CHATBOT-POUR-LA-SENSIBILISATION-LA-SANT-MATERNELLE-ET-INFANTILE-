# CHATBOT-POUR-LA-SENSIBILISATION-LA-SANT-MATERNELLE-ET-INFANTILE-
Ce chatbot a pour but de sensibiliser et d'informer les femmes enceintes et les jeunes mères  sur les bonnes pratiques de santé maternelle et infantile. Il peut répondre à des questions  sur la grossesse, les soins prénatals, la nutrition des enfants, et rappeler aux utilisateurs les dates de vaccination.
